📝 Условие задачиВам дан список интервалов, где каждый интервал представлен парой чисел: [start, end]. Некоторые интервалы пересекаются или соприкасаются. Напишите функцию merge_intervals(intervals), которая объединяет все пересекающиеся интервалы и возвращает новый список уже непересекающихся интервалов.📥 Входные и выходные данныеВход: [[1, 3], [2, 6], [8, 10], [15, 18]]Выход: [[1, 6], [8, 10], [15, 18]]Пояснение: Интервалы [1, 3] и [2, 6] пересекаются, поэтому они слились в один большой интервал [1, 6]. Остальные остались без изменений.Вход: [[1, 4], [4, 5]]Выход: [[1, 5]]
