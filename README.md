Новая задача: «Угадай число» (Мини-игра)Давайте закрепим работу с вводом данных (input) и условиями на чуть более игровой задаче.📝 УсловиеКомпьютер загадал число 7. Напишите программу, которая просит пользователя угадать это число.Пользователь вводит число через input().Если число равно 7, программа пишет: «Ура, ты угадал!».Если число меньше 7, программа пишет: «Загаданное число больше».
Если число больше 7, программа пишет: «Загаданное число меньше».💡 ПодсказкиЗагадайте число в переменную, например: secret_number = 7.Не забудьте превратить ввод пользователя в число с помощью int(), иначе Python будет сравнивать строку с числом и выдаст ошибку.
